Applications Docker

VSC

Installer la dernière version : 

docker run -it -p 80:8080 -v "/tmp/repos:/home/coder/project" codercom/code-server:latest --auth none

  • -it pour ouvrir la lecture et l'ecriture au terminal. Pas nécessaire
  • --auth none : Désactive l’autorisation pour se connecter. A supprimer si on veut la laisser
